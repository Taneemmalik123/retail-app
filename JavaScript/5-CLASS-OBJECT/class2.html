About Class & Object
  • class ? : The class is collections of MEMBER VARIABLE AND MEMBER function
  • Member variable ? The variables defined under constructure using "this" operator as this.customer this.product, this.service, the member variable are global variable of the class.
  • Member function ? All user defined function inside a class is called member function. as savecustomer(), getservice(), getproduct(), about()
  • constructor ? : The constructor is pre defined function of javascript class it is also called as life cycle function of javascript class. it get called automatically when object of a class get created.
  • this ? the "this" is a reserve keyword that work like object of current class. it is use to access member variable and member function but only inside the class
  • object ? the "object" is instance of a class in javascript
  • extends ? : the extends keyword use to inherit feature from 1 class to other
